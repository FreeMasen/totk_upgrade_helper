# Tears of the Kingdom Armor Helper

Track remaining required items for upgrading armor in the Legend of Zelda Tears of the Kingdom.

## Usage

Visit <https://freemasen.github.io/totk_upgrade_helper> and check the boxes for each item you've
upgraded to the appropriate level and the Item List table will update accordingly. All data is
stored in local storage so visiting on the same computer means you will be able to pick up where
you've left off.
